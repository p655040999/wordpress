== Description ==

Capital Titles Uppercase on your blog to allow Convert Lowercase titles to Upper Case in all post in your site.
This is a very simple and clean plugin.

== Installation ==

1. Upload `Capital_Titles_Uppercase.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done!
